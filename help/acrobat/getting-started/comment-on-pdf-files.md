# Comentar en un PDF

Aprenda a añadir comentarios en un PDF compartido utilizando un solo navegador web. Tanto si colaboras con un equipo como si estás revisando un documento, las herramientas de comentarios de Acrobat facilitan proporcionar comentarios claros y procesables. Este tutorial de vídeo utiliza la [nueva experiencia de Acrobat](new-experience.md).

Para aprender a compartir un PDF, echa un vistazo al tutorial de [Colaborar en tiempo real](collaborate.md).
