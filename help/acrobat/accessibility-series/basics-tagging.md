# Conceptos básicos del etiquetado

Aprende los aspectos básicos de cómo etiquetar un archivo de PDF para que sea accesible. Entre los temas tratados se incluyen: corrección de listas, creación de artefactos, adición de texto alternativo para imágenes y tablas simples.

>[!NOTE]
>
>Algunas funciones de este tutorial solo están disponibles en Acrobat Pro, Premium y Studio.
