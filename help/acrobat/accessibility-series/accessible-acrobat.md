# Crear un PDF accesible en Acrobat

Aprenda a crear un PDF accesible a partir de Acrobat. Los temas tratados incluyen: etiquetado automático basado en la nube, uso de las herramientas de preparación para la accesibilidad y solución de problemas comunes.

>[!NOTE]
>
>Algunas funciones de este tutorial solo están disponibles en Acrobat Pro, Premium y Studio.
